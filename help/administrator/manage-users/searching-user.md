    <br>&emsp;a. User type
    <br>&emsp;b. Medium
    <br>&emsp;c. Class
    <br>&emsp;d. Subject
    <br>&emsp;e. Block
    <br>&emsp;f. District
    <br>&emsp;g. School
    <br>&emsp;h. Role
    <br>4. Click <b>Submit</b> to apply the filters<br>Search results are displayed accordingly</td>
